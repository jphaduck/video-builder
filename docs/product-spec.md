# Product Spec

## Product name
Story Video Studio

## Core idea
A user enters a theme, premise, and optional plot notes. The app generates a YouTube-ready story video draft in slideshow format.

## Video format
- 5 to 20 minutes
- narrated story
- still-image-based scenes
- cinematic motion on images
- captions/subtitles
- music and sound design
- final MP4 export

## User flow
1. User creates a new project
2. User enters:
   - theme
   - premise
   - optional plot notes
   - target runtime
   - tone/style
3. App generates:
   - title options
   - hook
   - full script
   - scene-by-scene outline
4. App generates:
   - image prompts per scene
   - still images per scene
5. App generates:
   - voiceover
   - caption timing
6. App renders:
   - slideshow video with motion, transitions, subtitles, and audio
7. User reviews and edits:
   - script blocks
   - scene prompts
   - images
   - voice
   - timing
8. User exports final video

## Must-have features for v1
- new project creation
- premise/theme input
- script generation
- scene splitting
- image prompt generation
- still image generation
- voiceover generation
- captions
- timeline assembly
- final render
- project save/load
- manual regeneration for individual scenes

## Nice-to-have later
- thumbnail generation
- title/description variants
- chapter markers
- multiple visual styles
- multiple narration styles
- partial re-rendering
- per-scene motion presets

## Quality rules
- output should feel authored, not generic
- scenes should match narration
- images should feel stylistically consistent
- pacing should feel natural
- user should be able to revise weak outputs
